# Datathon Schneider Electric

# Presentació
Aquest es el repositori del repte de [`Schneider Electric`](https://github.com/data-students/datathon2025-challenges/tree/main/Schneider%20Electric) a la Datathon 2025. Realitzat per Martí Puig, Núria Pastor, Yunkai Chen i Sebastián Luna.

L’objectiu és entrenar un classificador binari per predir si una venda serà exitosa o no i, posteriorment, explicar com influeix cada característica en les prediccions.
